Запуск осуществляется из консоли cmd, вызванной из папки, где располагается jar'ник. (Или просто двойным кликом на иконку jar'ника, как с обычным .exe файлом)
Выполнить команду в консоли: java -jar X5Group_task.jar
Необходимо что бы файл inputData.txt находился в той же директории что и исполняемый jar'ник.
Данные ЗАХАРДКОЖЕНЫ, то есть при изменении входных данных, программа СЛОМАЕТСЯ!!!
Реализовано так в виду того, что в условии не написано обратное и плюс отсутствие времени на параметризацию.
Результат работы: Создастся Excel-файл с отформатированными данными из исходного файла.
